# 📊 E-Commerce Sales Analytics Dashboard using Excel & Power BI

## 🔎 Project Overview

This project analyzes e-commerce sales data for an electronics retailer to identify
revenue drivers, regional performance, and the financial impact of order cancellations.

The raw dataset was processed in Excel and visualized in Power BI to create an
interactive decision-support dashboard.

---

## 🛠 Tools Used

* Microsoft Excel (Data Cleaning, KPI Calculation, Data Structuring)
* Power BI (Dashboard Development & Interactive Visualizations)

---

## 📈 Key Business Metrics

* **Total Revenue:** ₹1.25 Billion
* **Total Orders:** 15.69K
* **Average Order Value (AOV):** ₹112.85K
* **Cancellation Rate:** 29.72%
* **Revenue Lost Due to Cancellations:** ₹525.4 Million

---

## 📊 Analysis Performed

* Product-level revenue comparison to identify top-selling items.
* Category-wise performance showing Laptops as primary revenue driver.
* State-wise sales analysis highlighting Maharashtra as the leading market.
* Quarterly trend analysis to monitor revenue fluctuations across 2024.
* KPI tracking for operational efficiency and loss monitoring.

---

## 🎯 Objective

To demonstrate real-world data analytics workflow:
Data Cleaning → KPI Development → Visualization → Business Insight Generation.
